RISC-V Computer Architecture Workshop

2023/03/07(火),08(水)に慶應義塾大学で開催されたRISC-V Computer Architecture Workshopに参加しました

RISC-V Computer Architecture Workshop

Gitefu

2023-23-03

研究室のメンバー数名でRISC-Vコンピューター アーキテクチャ ワークショップ(RISC-V Computer Architecture Workshop)に参加いたしました．

会場ではFPGA評価ボード(Nexys A7とNexys DDR)が配布され， コンピュータ・アーキテクチャやRISC-V命令セット・アーキテクチャについて実際にFPGAで動かしながら学ぶこことができました．

また，会場にはお菓子やジュースが用意されており，ゆるい雰囲気でとてもリラックスして参加できるワークショップでした．

RISC-V Computer Architecture Workshop 「RVfpga: コンピューター アーキテクチャを理解する対面ワークショップ」

https://riscv.or.jp/2023/01/rvfpga_workshop7th_8th_march_23/

Imagination Technology / Imagination University Program
